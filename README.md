# Overframe Mobile Portal

Un portal web ligero y adaptado a móviles para acceder rápidamente a la información de [Overframe.gg](https://overframe.gg).

## Características

*   **Diseño Premium App-like:** Interfaz oscura con efectos visuales estilo Warframe, animaciones fluidas y soporte para modo oscuro.
*   **Progressive Web App (PWA):** Instálalo en tu móvil para usarlo como una aplicación nativa, con icono propio y funcionamiento a pantalla completa.
*   **Búsqueda Rápida:** Busca Warframes, Armas y otros items directamente desde la pantalla principal.
*   **Accesos Directos:** Botones grandes y accesibles para la Tier List y Top Builds.
*   **Categorías:** Navegación rápida a las secciones principales de Overframe.

## Cómo usar

### Opción 1: Ejecutar localmente
Simplemente abre el archivo `index.html` en tu navegador web (Chrome, Safari, Firefox).

### Opción 2: Hosting (Recomendado)
Sube los archivos (`index.html`, `style.css`, `script.js`, `manifest.json`, `sw.js`, iconos) a un servicio de hosting con HTTPS (necesario para PWA) como GitHub Pages, Vercel, o Netlify.

### Instalar como App
1.  Abre el portal en tu navegador móvil (Chrome en Android, Safari en iOS).
2.  Abre el menú del navegador.
3.  Selecciona **"Añadir a pantalla de inicio"** o **"Instalar aplicación"**.
4.  El portal aparecerá en tu lista de apps con su propio icono.
